The function LargeImg() is triggered when the user mouse over the image.
This function enlarges the image. This is a onmouseover effect,
the onmouseover attribute is firing whenever the mouse pointer moves over the element.

The function regularImg() is triggered when the mouse pointer is moved out of the image.
That function sets the height and width of the image back to normal. This is a onmouseout effect,
the onmouseout attribute is firing whenever the mouse pointer moves out of an element.

This example demonstrates how to assign an "onmouseover" and "onmouseout" event to a h1 element.
